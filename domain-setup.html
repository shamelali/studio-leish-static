Studio Leish

Domain Setup Guide

Domain: studio@leish.my

1 Cloudflare Setup

If you haven't already, add leish.my to Cloudflare:

https://dash.cloudflare.com → Add a Site → leish.my

After adding, Cloudflare will provide nameservers like:
- merlin.ns.cloudflare.com
- vivienne.ns.cloudflare.com
Update these at your domain registrar (MYNIC, Exabytes, etc.)

2 DNS Configuration

Add CNAME record for studio subdomain:

Type: CNAME
Name: studio
Content: cname.vercel-dns.com
Proxy: Enabled (orange cloud)
TTL: Auto

Do this in Cloudflare dashboard → DNS → Records → Add Record

3 Vercel Domain

Add domain in Vercel dashboard:

vercel domains add studio.leish.my

Or go to: https://vercel.com/shamelalis-projects/studio-leish-static/settings/domains
Click "Add Domain" and enter: studio.leish.my

4 SSL/TLS Configuration

Set encryption mode in Cloudflare:

Cloudflare → SSL/TLS → Overview
Set to: Full (strict)

Also enable: Always Use HTTPS, HTTP Strict Transport Security (HSTS)

⏳ DNS Propagation Time:
After setting up DNS, wait 24-48 hours for full propagation.
You can check status with: dig studio.leish.my

Verification

Test URLs (after propagation):

Homepage: https://studio.leish.my
Booking: https://studio.leish.my/book.html
Admin: https://studio.leish.my/admin.html
User: https://studio.leish.my/user.html

Update Application URLs:

After domain is live, update these in your code:

In book.html:
- from: 'Studio Leish <bookings@leish.my>'
- callback_url: 'https://studio.leish.my/api/billplz-callback'
In admin.html & user.html:
- Update redirect URLs to: 'https://studio.leish.my/...'
